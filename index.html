Todo o período

Configurações

Personalize o aplicativo, gerencie acessos e configure a sincronização com GitHub.

Logo
A logo será convertida em Base64 e salva no arquivo JSON do GitHub.
QR Code
A imagem do QR Code será salva no GitHub fora do JSON. O JSON guarda somente o caminho do arquivo.

Configuração GitHub

Os dados serão salvos em um arquivo JSON dentro do seu repositório.

Atenção: o token GitHub é salvo somente neste navegador. Não publique token dentro do código.

Configurações de acesso

Adicione, edite ou remova usuários que podem acessar o sistema.

Aviso de segurança: este login é local no front-end. As senhas são salvas como hash SHA-256 no arquivo JSON.
Ao editar, deixe em branco para manter a senha atual.
Usuário somente leitura não consegue salvar, editar ou remover dados.

Usuários cadastrados

O usuário padrão pode ser editado após o primeiro acesso.

Backup local

Além do GitHub, você pode exportar uma cópia local em JSON.
